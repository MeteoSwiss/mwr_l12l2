API
===

The core functionality of this API happens in the following modules. *Retrieval* orchestrates the retrieval and
takes care of pre- and post-procesing. *Models" is there to request and interpret input data from Numerical Weather
Prediction models. The *Utils* module contains small functions to assist the other modules.
